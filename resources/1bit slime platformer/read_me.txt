> INFO

All tiles are 8x8 pixels.
The only exception being the water ripple, that needs 24px minimum for the animation to work.

full_tileset character animation by row: idle; walking; death; jump

> SUGESTIONS

3FPS for animations with 2 sprites and 6FPS for animations with 4 sprites, the water_fall and the water_ripple.

water_fall can be used in any size tiles. Always start from the last row, and repeat the pattern for a bigger animation if necessary. 

water_ripple can be cut. Only needing 24px wide minimum. Also, use the first sprite to place a normal water tile.

slime2 as an enemy and drop_icon as slime lives.
